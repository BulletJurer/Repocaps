[Para maquina Madre/local]
Instalar node.js
npm install express
-en el archivo server.js esta el nro de puerto a configurar junto con el directorio a usar
npm install -g localtunnel
npm install -g nodemon

[iniciacion]

-Localtunel desde powershell tiene que ejecutar el webhook permanente
lt --port 7777 subdomain repocapingerhook

-Este comando tiene que ser ejecutado en el directorio del proyecto(cd webing)
nodemon server.js

-revelar pagina al internet con localtunnel:
lt --port 3000 subdomain testeorepocaps
